## Development Methodology
- **TDD (Test-Driven Development) 필수**
  - 코드를 먼저 짜지 않는다. 반드시 테스트를 먼저 작성한다.
  - Red → Green → Refactor 사이클을 준수한다.
  - 새 기능 추가 또는 기존 코드 변경 시, 기존 테스트 전체를 실행하여 회귀(regression)를 방지한다.
